Workshop 1.1: การประกาศตัวแปรและการแสดงผลพื้นฐานเป้าหมาย: ฝึกการสร้างตัวแปรเพื่อเก็บข้อมูลส่วนตัว (ชื่อ, อายุ, สาขา).การทำงาน: * ใช้คำสั่ง const หรือ let เพื่อจองพื้นที่ในหน่วยความจำสำหรับเก็บค่า.ใช้คำสั่ง console.log() เพื่อนำค่าจากตัวแปรออกมาแสดงบนหน้าจอ Console.มีการใส่ Comment (ข้อความหลัง //) เพื่ออธิบายโค้ดโดยที่โปรแกรมจะไม่นำส่วนนี้ไปประมวลผล.
Workshop 1.2: ประเภทข้อมูล (Data Types) และการตรวจสอบเป้าหมาย: ทำความเข้าใจความแตกต่างของข้อมูลแต่ละประเภทใน JavaScript.การทำงาน:สร้างตัวแปร 5 ชนิด ได้แก่ String (ข้อความ), Number (ตัวเลข), Boolean (ค่าจริง/เท็จ), Null (ค่าว่าง), และ Undefined (ค่าที่ยังไม่ได้กำหนด).ใช้ Operator typeof นำหน้าชื่อตัวแปร เพื่อให้โปรแกรมส่งค่าชนิดของข้อมูลนั้นกลับมาเป็นข้อความ.ข้อควรจำ: typeof null จะคืนค่าเป็น "object" ซึ่งเป็นลักษณะเฉพาะของภาษา JavaScript.
Workshop 1.3: การคำนวณและตัวดำเนินการเปรียบเทียบเป้าหมาย: ใช้ตัวดำเนินการทางคณิตศาสตร์และการเปรียบเทียบค่า.การทำงาน:การคำนวณ: นำตัวแปรคะแนนมาบวกกัน (+) เพื่อหาผลรวม และใช้การหาร (/) เพื่อหาค่าเฉลี่ย.สูตรการคำนวณค่าเฉลี่ยคือ: $Average = \frac{Math + Science + English}{3}$การเปรียบเทียบ: ใช้เครื่องหมาย >= (มากกว่าหรือเท่ากับ) เพื่อเช็คเงื่อนไข ผลลัพธ์ที่ได้จะเป็นประเภท Boolean เสมอ (true หรือ false).
Workshop 1.4: ระบบตรวจสอบสิทธิ์ (Control Flow)เป้าหมาย: ฝึกการใช้เงื่อนไข (Conditional Logic) ในการตัดสินใจของโปรแกรม.การทำงาน:ใช้โครงสร้าง if...else if...else เพื่อแยกทิศทางการทำงาน.ใช้ Logical Operator && (AND) เพื่อรวมเงื่อนไขเข้าด้วยกัน (เช่น ต้องถูกทั้งชื่อ, รหัส และอายุถึงจะผ่าน).ลำดับการตรวจสอบ: โปรแกรมจะเช็คเงื่อนไขแรกก่อน หากไม่ตรงจะข้ามไปเช็ค else if และหากไม่ตรงเลยจะตกไปที่ else ตัวสุดท้าย.
